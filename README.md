# 🐾 LuxuryPet - Banho e Tosa Premium

Uma landing page elegante e moderna para um PetShop de alto padrão, desenvolvida com foco na experiência do usuário (UX) e na praticidade de agendamento de serviços estéticos para cães e gatos.

---

## ✨ Demonstração Visual e Estilo

O projeto conta com um design exclusivo **Dark Neon**, combinando sofisticação e modernidade. 
* **Cores principais:** Grafite escuro (`#121214`), Preto fosco (`#1a1a1e`) e tons vibrantes de Ciano Neon (`#00f5d4`) para botões e destaques.
* **Animações:** Efeitos de *pulse* em elementos de conversão (CTA) e transições suaves nos cards de serviços.
* **Responsividade:** Interface 100% adaptável para smartphones, tablets e computadores.

---

## 🚀 Funcionalidades Principais

* **Navegação Fluida:** Menu de cabeçalho fixo (*sticky*) com rolagem suave (*smooth scroll*) para as seções.
* **Vitrine de Serviços:** Cards interativos detalhando os preços, tempos estimados e especificações de cada tratamento, com destaque para o serviço "Mais Vendido".
* **Sistema de Agendamento Inteligente:**
    * **Bloqueio de Datas Passadas:** O calendário não permite selecionar datas anteriores ao dia atual.
    * **Máscara de Telefone:** Formatação automática do input de contato enquanto o usuário digita: `(00) 00000-0000`.
    * **Validação de Campos:** Verificação nativa via JavaScript para garantir que nenhum dado essencial seja enviado em branco ou incompleto.
    * **Feedback customizado:** Exibição de alertas personalizados de sucesso ou erro integrados diretamente no layout.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído puramente com tecnologias web fundamentais (Vanilla Architecture):

* **HTML5:** Estruturação semântica de seções, formulários e acessibilidade.
* **CSS3:** Estilização baseada em variáveis (`:root`), Flexbox, CSS Grid para os cards, animações com `@keyframes` e Media Queries para responsividade.
* **JavaScript (ES6):** Manipulação dinâmica do DOM, validação de dados, escuta de eventos (`addEventListener`) e lógica de negócio do formulário.
* **Google Fonts:** Utilização da tipografia moderna *Inter*.

---

## 💻 Como Executar o Projeto

Como o projeto foi desenvolvido com tecnologias nativas do navegador, não é necessário instalar dependências.

1. Faça o clone ou baixe os arquivos deste repositório.
2. Certifique-se de que os arquivos `index.html`, `style.css` e `script.js` estejam na mesma pasta.
3. Abra o arquivo `index.html` diretamente em qualquer navegador web (Chrome, Edge, Firefox, Safari) ou utilize a extensão **Live Server** no VS Code.
